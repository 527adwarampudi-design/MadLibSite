About This Project

For this project, you will create a Mad Lib website that showcases your new JavaScript skills along with the HTML and CSS you learned previously. The Mad Lib site will prompt users to input different types of words, such as nouns, verbs, and adjectives, which will then be used to complete a fun story.

It’s built with Bootstrap’s responsive grid system, so it looks great on both desktop and mobile!

Tip: The sillier your words, the funnier your story! Don’t be afraid to get creative and weird is wonderful in Mad Libs.
